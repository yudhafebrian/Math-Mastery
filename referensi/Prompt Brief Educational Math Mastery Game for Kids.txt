# Math Fact Mastery MVP

You are a senior React developer, product designer, educational game designer, and software architect.

Build a simple MVP for a math fact practice game for elementary school children.

The goal is not to create a commercial learning platform.

The goal is to help one child build automatic recall of basic addition and multiplication facts.

Keep the implementation simple.

Prefer simplicity over completeness.

Avoid over-engineering.

---

# Core Goal

Help children memorize arithmetic facts until recall becomes automatic.

Track:

* Accuracy
* Response Time

Show parents which facts are strong and which facts still need practice.

# Learning Philosophy

Children should learn skills in sequence.

A child should not move to harder skills before mastering the foundation.

Because of this, skill unlocks are required.

# Important Rule

Do NOT merge commutative facts.

Treat:

6+7

and

7+6

as separate facts.

Treat:

7×8

and

8×7

as separate facts.

Track them independently.

---

# Skill Unlock

A skill becomes unlocked when:

Accuracy >= 90%

AND

Average Response Time <= 3 seconds

for the current skill.

Only then may the next skill be practiced.

---

# Question Modes

Only implement:

1. Fill In Blank
2. Multiple Choice
3. True / False

Do not implement any other game modes.

---

# Question Recording

For every answer save:

child_id

fact_id

is_correct

response_time_ms

created_at

---

# Adaptive Practice

Facts that are frequently wrong should appear more often.

Facts with slow response times should appear more often.

Facts that are consistently correct and fast should appear less often.

Keep the algorithm simple.

No advanced mastery formulas required.

---
